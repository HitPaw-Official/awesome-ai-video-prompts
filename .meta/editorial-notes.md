# Editorial Notes

This project should present prompt examples, templates, and writing guidance in a user-facing tone.

## Writing principles

- Focus on what users can create or learn
- Keep model names and prompt types clear
- Use natural invitation language when linking to HitPaw
- Avoid internal strategy language in public-facing content
- Avoid describing keyword, ranking, CTA, or conversion strategy in public docs

## Public-facing tone

Good examples:
- Explore Seedance prompt ideas for cinematic scenes
- Try this prompt in HitPaw AI Video Generator
- Use this template to build your own text-to-video prompt

Avoid in public docs:
- capture search intent
- rank for broad terms
- long-tail keyword strategy
- CTA strategy
- conversion path
