# Seedance Prompt Templates

## Basic cinematic formula
Subject + action + scene + lighting + camera movement + mood

Example:
A detective lights a cigarette in a narrow alley, wet pavement, moody blue lighting, slow camera push in, noir cinematic tone.

## Character motion formula
Character + movement + environment + visual style + shot type

Example:
A young explorer runs across ancient stone ruins, jungle environment, realistic cinematic style, wide tracking shot.

## Fantasy formula
Character or creature + magical action + fantasy setting + atmosphere + color palette

Example:
A glowing dragon circles above a crystal castle, twilight sky, mystical fog, blue and violet palette, epic fantasy mood.
