# AI Video Prompt Formulas

## Universal prompt formula
Subject + action + scene + camera + lighting + mood

## Cinematic formula
Character or object + motion + environment + cinematic shot + dramatic lighting + tone

## Product ad formula
Product + feature highlight + visual setting + camera motion + premium ad style

## Anime formula
Character + expressive movement + stylized environment + anime visual language + emotional tone

## Social short video formula
Subject + hook action + bold scene setup + fast visual cue + platform-friendly energy

## Why formulas matter

Prompt formulas reduce randomness and make it easier to scale content across multiple model hubs.
